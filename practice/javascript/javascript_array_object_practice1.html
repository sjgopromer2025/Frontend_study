자바스크립트 배열과 객체 실습 - 게임 캐릭터와 인벤토리 관리

콘솔을 열어 결과를 확인하세요!

실습 조건 설명

  • 캐릭터 정보 객체: 캐릭터의 이름, 레벨, 직업, 체력, 마나, 인벤토리를 포함합니다.
  • 아이템 추가 함수: 새로운 아이템을 인벤토리에 추가합니다.
  • 아이템 제거 함수: 특정 아이템을 인벤토리에서 제거합니다.
  • 레벨 업 함수: 캐릭터의 레벨을 올리고 체력과 마나를 증가시킵니다.
  • 아이템 확인 함수: 특정 아이템이 인벤토리에 있는지 확인합니다.

현재 캐릭터 상태

  • 이름: 용사
  • 레벨: 11 (레벨 업 후)
  • 직업: 전사
  • 체력: 120 (레벨 업 후)
  • 마나: 60 (레벨 업 후)
  • 인벤토리: 검, 방패, 활
