자바스크립트 배열과 객체 실습 - 쉬운 버전

콘솔을 열어 결과를 확인하세요!

실습 조건 설명

  • 캐릭터 정보 객체: 캐릭터의 이름, 레벨, 직업, 체력, 인벤토리를 포함합니다.
  • 아이템 추가 함수: 새로운 아이템을 인벤토리에 추가합니다.
  • 레벨 업 함수: 캐릭터의 레벨을 올리고 체력을 증가시킵니다.
  • 인벤토리 확인 함수: 현재 인벤토리를 출력합니다.

현재 캐릭터 상태

  • 이름: 초보자
  • 레벨: 2 (레벨 업 후)
  • 직업: 모험가
  • 체력: 60 (레벨 업 후)
  • 인벤토리: 나무검, 포션, 방패
