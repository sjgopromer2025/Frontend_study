자바스크립트 함수 실습 - 게임 캐릭터 관리

콘솔을 열어 결과를 확인하세요!

함수 조건 설명

  • 캐릭터 인사 함수: 캐릭터 이름을 받아 "안녕하세요, 저는 [이름]입니다!"를 반환합니다.
  • 경험치 계산 함수: 현재 경험치와 추가 경험치를 받아 총 경험치를 반환합니다.
  • 캐릭터 정보 출력 함수: 이름, 레벨, 직업을 받아 캐릭터 정보를 반환합니다.
  • 체력 회복 함수: 현재 체력과 회복량(기본값: 20)을 받아 회복된 체력을 반환합니다.
  • 아이템 목록 출력 함수: 나머지 매개변수를 사용하여 획득한 아이템 목록을 반환합니다.
  • 공격력 계산 함수: 기본 공격력과 추가 공격력을 받아 총 공격력을 반환합니다.

현재 게임 상태

  • 캐릭터 이름: 용사
  • 레벨: 10
  • 직업: 전사
  • 현재 체력: 50 (회복 후: 70)
  • 획득한 아이템: 검, 방패, 포션
  • 총 공격력: 50
