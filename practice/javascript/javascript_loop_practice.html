반복문 실습 - 게임 아이템 수집

콘솔을 열어 결과를 확인하세요!

반복문 조건 설명

  • for 반복문: 아이템을 1부터 5까지 수집합니다.
  • while 반복문: 체력이 100이 될 때까지 20씩 회복합니다.
  • do-while 반복문: 보스를 공격하여 체력이 0이 될 때까지 반복합니다.
  • for...of 반복문: 인벤토리의 모든 아이템을 출력합니다.
  • for...in 반복문: 캐릭터의 속성과 값을 출력합니다.

현재 게임 상태

  • 수집한 아이템: 검, 방패, 포션, 활
  • 현재 체력: 100
  • 보스 상태: 처치 완료
  • 캐릭터 정보: 이름: 용사, 레벨: 10, 직업: 전사
