반복문 예제

콘솔을 열어 결과를 확인하세요!

반복문 설명

  • for 반복문: 특정 횟수만큼 반복 실행합니다. 예: 0부터 4까지 출력.
  • while 반복문: 조건이 참인 동안 반복 실행합니다. 예: 카운트가 3보다 작을 때까지 출력.
  • do-while 반복문: 최소 한 번은 실행한 후 조건을 확인합니다. 예: 숫자가 2보다 작을 때까지 출력.
  • for...of 반복문: 배열의 각 요소를 순회합니다. 예: 과일 목록 출력.
  • for...in 반복문: 객체의 속성을 순회합니다. 예: 객체의 키와 값을 출력.
