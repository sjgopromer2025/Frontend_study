함수 개념 설명

  • 함수 선언문: `function` 키워드를 사용하여 함수를 선언합니다.
  • 함수 표현식: 함수를 변수에 할당하여 표현합니다.
  • 매개변수와 인자: 함수는 매개변수를 받아 호출 시 인자를 전달받아 동작합니다.
  • 반환값: 함수는 `return` 키워드를 사용하여 값을 반환할 수 있습니다.
  • 함수 호출: 함수 이름 뒤에 괄호를 붙여 호출합니다.
  • 스코프: 함수 내부에서 선언된 변수는 함수 외부에서 접근할 수 없습니다.
  • 즉시 실행 함수 (IIFE): 선언과 동시에 실행되는 함수입니다.
  • 기본값: 매개변수에 기본값을 설정할 수 있습니다.
  • 나머지 매개변수: `...`을 사용하여 여러 인자를 배열로 받을 수 있습니다.
  • 화살표 함수: 간결한 문법으로 함수를 작성할 수 있습니다.
